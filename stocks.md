## How does it work

First, let us consider the momentum factor. This factor means that
stocks that have performed well in the past tend to continue doing so
in the future. Imagine that you have a stock that goes straight up
for several months. This stock will be included in various ETFs that
target the momentum factor, such as *iShares Edge MSCI USA Momentum
Factor ETF*. The more momentum ETFs include a particular stock, the
more likely it is that this stock is a good momentum under different
implementations and assumptions.

Similarly, we can do the same for other factors: quality, value and
low volatility.  Every [ETF](etf.md) is categorized in zero, one or
more of these factors. To identify which stocks have which factors,
we can count how many ETFs with each factor (or with any factors)
hold that stock. The table above contains the ranking of the stocks
in each factor, and aggregated. The counts are not shown because
they are not directly comparable across different factors.
